## üì• Installation

```bash
# Clone repository
git clone https://github.com/ashintv/NexMeet
cd NexMeet

# Backend (Express Server)
cd server
npm install
npm run dev        # Development
# OR
npm run build && npm run start   # Production

# WebSocket Server
cd ../Websocket-server
npm install
npm run dev        # Development
# OR
npm run build && npm run start   # Production

# Frontend (React + Vite)
cd ../NexApp
npm install
npm run dev

# Access frontend at http://localhost:5173
